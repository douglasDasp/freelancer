3. Circuito de Alimentação Sugerido
Para carregar a bateria e alimentar o circuito com segurança no menor espaço possível, utilize um módulo TP4056 com proteção:

Bateria LiPo (3.7V) -> Conectada ao TP4056.

Saída do TP4056 -> Conectada ao pino 3V3 do ESP32 (ignorando o regulador interno se a bateria estiver estabilizada em 3.3V via LDO externo) ou ao pino VIN/5V (menos eficiente).

4. Plano de Ação para Código (Firmware)
Para atingir essa autonomia, seu código deve seguir esta lógica:

Deep Sleep Total: O ESP32 fica desligado.

Wake-up Source: Use um botão físico ou o pino de interrupção (IRQ) do PN532 para acordar o ESP32.

Leitura Rápida: Acorda -> Lê a Tag -> Armazena na memória Flash/SPIFFS -> Volta a dormir.

Consulta BLE: Somente quando um botão de "Sync" for pressionado, o BLE é ativado para você ler os IDs pelo celular.

Resumo do Hardware ideal:
Bateria: LiPo 1000mAh (Equilíbrio perfeito entre volume e carga).

Carregador: Módulo micro-USB TP4056 (Pequeno e barato).

Regulador (Opcional): Um LDO de baixo consumo como o MCP1700-3302E para substituir o original da placa.
